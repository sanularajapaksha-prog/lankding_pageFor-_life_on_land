Home Page

Technical Description

I designed the Home page to offer a user-friendly introduction to the Life on Land initiative. The page uses HTML5 semantic tags like <header>, <main>, and <footer>. Visuals like a background banner, welcoming headline, and call-to-action buttons are used to encourage user interaction.

CSS techniques like flexbox and media queries were applied for responsive design. A background video and overlay were implemented for immersive engagement.

Accessibility

Text content follows proper contrast ratios and includes semantic headings. All images and icons have appropriate alt attributes. The navigation is fully keyboard accessible.

Table Page

Technical Description

The Table page displays biodiversity and sustainability metrics using an interactive and accessible HTML table. Columns include Country Name, Reforestation %, Endangered Species, and Clean Energy Usage. I used semantic elements like <table>, <thead>, and <caption>.

To enhance readability, CSS was used to style alternating row colors, borders, and hover highlights. A background video adds depth, while transparent sections maintain focus on data.

Accessibility

The table includes headers (<th>) and scope attributes to support screen readers. All colors were tested for color contrast, and the page remains navigable via keyboard only.

Content Page

Technical Description

This content page, titled “ Biodiversity” was developed using HTML <article> and <section> tags to present facts and insights about deforestation, wildlife, and reforestation. Icon-enhanced headings and fact boxes were styled with CSS grid and custom icons from Font Awesome.

Each content block includes an image, a descriptive heading, and fact-based text for better user engagement. Hover and transition effects were used to make interactions smoother.

Accessibility

All visual content has descriptive alt text. The layout is fully responsive, with accessible font sizes and high contrast. All headings follow a proper semantic hierarchy.

Challenges and Lessons Learned

One challenge was integrating a video background without slowing the page load. I resolved this by compressing the video and using lazy loading techniques. Another challenge was building responsive tables. I overcame it by restructuring with flexbox and hiding low-priority columns on mobile devices.

Through this, I learned the importance of semantic HTML, accessibility, and testing across devices and browsers.

Compliance

All pages meet the Janet regulations for accessibility, clean HTML, and responsiveness. I validated all pages using W3C tools and ensured alt text, form label linking, and proper use of semantic elements.
